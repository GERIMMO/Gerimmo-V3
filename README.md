# GERIMMO V3

Socle applicatif GERIMMO V3 base sur Next.js, TypeScript, Tailwind CSS v4 et shadcn/ui.

Cette base conserve le layout dashboard, la sidebar, le header, les themes et les composants UI reutilisables de la template d'origine. Les pages de demonstration sont masquees de la navigation afin de laisser une structure metier propre pour les prochaines etapes.

## Modules prepares

- Accueil
- Incidents
- Biens
- Locataires
- Proprietaires
- Artisans
- Documents
- Echanges
- Notifications
- Rapports
- Parametres
- Super Admin

## Commandes

```bash
npm install
npm run dev
npm run build
```

## Notes

Supabase, l'authentification, les roles et la logique metier ne sont pas encore connectes. Ces sujets sont reserves aux prochaines etapes d'architecture.
